# Iris Flower Classification: Model Comparison

## Overview

This project explores and compares multiple machine learning classification algorithms on the famous Iris Flower Dataset. The goal is to classify iris flowers into one of three species based on their petal and sepal measurements.

The project demonstrates a complete machine learning workflow including data preprocessing, model training, cross-validation, and performance evaluation.

---

## Dataset

The Iris dataset contains 150 samples of iris flowers belonging to three species:

* Iris Setosa
* Iris Versicolor
* Iris Virginica

Features used:

* Sepal Length
* Sepal Width
* Petal Length
* Petal Width

Target Variable:

* Species

---

## Objectives

* Perform multiclass classification on the Iris dataset
* Compare the performance of multiple machine learning algorithms
* Apply data preprocessing techniques
* Evaluate models using multiple metrics
* Evaluated model performance using 5-Fold Cross-Validation

---

## Machine Learning Models Used

### 1. Logistic Regression

A linear classification algorithm that estimates class probabilities using the logistic function.

### 2. Gaussian Naive Bayes

A probabilistic classifier based on Bayes' theorem with the assumption of feature independence.

### 3. K-Nearest Neighbors (KNN)

A distance-based algorithm that classifies samples according to their nearest neighbors.
